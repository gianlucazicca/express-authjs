# Express Auth.js

## Installation
install dependencies:

```bash
pnpm install
```

## Usage

### Building

To build the project:

```bash
pnpm run build
```

### Starting

To start the server:

```bash
pnpm start
```

### Development

For development with hot-reloading:

```bash
pnpm run dev
```

### Testing

To run tests:

```bash
pnpm test
```
